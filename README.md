# Saffire

**Saffire is a progressive web application that allows users to create itineraries and share them with their travel group.**

<img width="1280" alt="saffire_main" src="https://user-images.githubusercontent.com/18601434/31747783-d3a130e4-b43c-11e7-9796-5a94e18624a3.png">



**Being a progressive web application, it allows users to use Saffire directly from the browser. Utilizing a responsive design, it enables the app to be accessed on the users' browsers and on their mobile devices**



<img width="1280" alt="saffire_allitin" src="https://user-images.githubusercontent.com/18601434/31747782-d398a33e-b43c-11e7-895e-9095a352beaf.png">



**Users can send status notifications, search for places, and add URL links for events they are interested in. Friends who are added to the itinerary can also make their own suggestions and leave comments to help with the itinerary decision**



<img width="1280" alt="saffire_ideaboard" src="https://user-images.githubusercontent.com/18601434/31747780-d38887a6-b43c-11e7-85da-a072eccee9b5.png">



**Users can see a map with all their points of interest, create a schedule for each event or location, and click on a location to get more details about it**
